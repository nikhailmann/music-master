Music Master

How well do you know music? This application uses the Spotify API to answer that question. Each round, two artists will be displayed on the screen, and you must guess which one is more popular according to Spotify's calculated "popularity" index (a 0-100 score).

To play, you will need to create a free API login by going to Spotify's developer dashboard and logging in with your spotify account. From there, you must create an app with a title and a description (put anything you want there). Once created, click on the app, and there you will see your Client ID and can press a button to reveal your Client Secret. Enter them here and submit to begin!
